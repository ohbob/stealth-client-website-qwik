Ignore messages from selected mobile (means messaged will not be shown in journal and can be taken by script methods).

**NB**: Not actual for connected clients.


### Params:
  *Mobile* : Name of mobile to be ignored. 

 ### DWS\Pascal Script:

```pascal

```
